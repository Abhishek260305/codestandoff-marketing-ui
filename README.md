# CodeStandoff Marketing UI

Marketing page for CodeStandoff 2.0.

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3006](http://localhost:3006) with your browser to see the result.

## Tech Stack

- Next.js 15.5.7
- TypeScript
- Tailwind CSS
- React 18.3.1
